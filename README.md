# SolarNow

#### 介绍
基于开源教程TinyRender，利用CPU渲染并写入序列帧，图片的转换和播放皆由ffmpeg完成。

可以展示旋转的obj模型，可用于简单obj文件的快速预览。

#### 使用
将obj和png放入model文件夹，只要保证两者同名即可。然后运行SolarNow.exe，即可弹出预览窗口。

按空格可以暂停旋转，然后鼠标邮件按住可以左右拖动。
